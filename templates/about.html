About This Project

Project Overview

This project presents a web-based image forgery detection system designed for the era of AI-generated content by integrating traditional forensic techniques with neural forensic methods.

In the digital era, images play a crucial role in social media, journalism, legal documentation, and online communication. With the rapid advancement of artificial intelligence, modern image generation tools have made it possible to create highly realistic synthetic and manipulated images. This has significantly increased the risks of misinformation, digital fraud, and misuse of visual evidence.

Project Objectives
  • Develop a browser-based image forgery detection system
  • Apply Error Level Analysis (ELA) for compression-based anomaly detection
  • Extract statistical features (mean intensity, variance, standard deviation) from ELA outputs
  • Detect Photoshop-based manipulations including noise inconsistencies, cloning/copy-move regions, edge artifacts, and color blending anomalies
  • Classify images as Original or Morphed/Edited using rule-based, interpretable logic
  • Generate forensic visualizations and explanations understandable to general users
  • Support lightweight, non-machine learning forensic workflows suitable for academic use
