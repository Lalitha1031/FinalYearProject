Error Level Analysis (ELA)

ELA is a forensic technique that identifies areas of an image with different compression levels:

  • Original image is resaved at a known quality (95%)
  • Difference between original and resaved version is calculated
  • Tampered regions show different error levels
  • Results are amplified for visualization
Key Insight: Authentic images show uniform error levels, while edited regions exhibit inconsistent compression patterns.
Statistical Analysis

Multiple statistical features are extracted from the ELA output:

  • Mean Intensity: Average error level across image
  • Variance & Std Dev: Spread of error values
  • Local Variance: Block-wise inconsistency
  • Edge Intensity: Error levels at boundaries
Threshold-Based Detection: Values exceeding predefined thresholds indicate potential manipulation.
Manipulation Detection
Four Key Checks:
  1. Noise Inconsistency: Edited regions have different noise characteristics than original areas
  2. Clone Detection: Identifies duplicated regions (copy-move forgery) using template matching
  3. Edge Anomalies: Detects unnatural sharp boundaries from cut-paste operations
  4. Color Blending: Finds halo effects and color mismatches around pasted objects
Classification Logic

Rule-based decision making:

Manipulated ≥3 anomalies detected
Likely Manipulated 2 anomalies detected
Possibly Manipulated 1 anomaly detected
Authentic No anomalies detected

Confidence scores are calculated based on the number and severity of detected anomalies.

Limitations
